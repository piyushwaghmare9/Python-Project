📚 PCCOE Prep
A comprehensive, interactive web application designed to help engineering students practice Previous Year Questions (PYQs) and prepare for their exams. Built as a fast, responsive Single Page Application (SPA), it offers chapter-wise quizzes, bookmarking capabilities, and direct access to study materials.

✨ Features
Interactive Mock Quizzes: Chapter-wise question banks with a clean, distraction-free interface.
Real-time Feedback: Instant answer checking with visual cues for correct (green) and incorrect (red) responses.
Smart Bookmarking: Save difficult questions to your bookmarks for quick revision. Bookmarks are persistently saved in your browser using localStorage.
Progress Tracking: Visual progress bars and a question-grid map to easily track attempted, unseen, correct, and wrong answers.
Download Hub: Dedicated section with direct Google Drive links to download full PYQ PDFs, formula sheets, and answer keys.
Theme Toggle: Built-in Light and Navy (Dark) modes for comfortable reading in any environment.
Responsive Design: Fully optimized for both desktop and mobile devices.

📖 Subjects Covered
The platform currently includes meticulously categorized question banks for the following core engineering subjects:
Linear Algebra & Univariate Calculus (Matrices, Differential Calculus, ODEs, etc.)
Engineering Chemistry (Water Technology, Fuels, Corrosion, Polymers)
Digital Electronics & Computer Organization (Logic Gates, Sequential Logic, Architecture)
Computer Programming & Problem Solving (C Programming, Arrays, Sorting, Data Structures)

🛠️ Tech Stack
This project is built using pure, vanilla web technologies. No external frameworks or libraries are required, making it incredibly fast and easy to deploy.
HTML5: Semantic structure and SPA layout.
CSS3: Custom variables, flexbox/grid layouts, and responsive media queries.
JavaScript (Vanilla): DOM manipulation, state management, scoring logic, and localStorage integration.

🚀 Getting Started
Since this is a client-side only application, setting it up is instant.

Prerequisites
A modern web browser (Chrome, Firefox, Safari, Edge).

Installation
Clone the repository or download the source code ZIP.

Bash
git clone https://github.com/yourusername/pccoe-prep.git
Navigate to the project folder.

Open the index.html file directly in your web browser.

💻 How to Use
Login: Enter any username on the welcome screen to enter the app (currently a mock login for personalization).
Select a Subject: Navigate to the "Subjects" tab and choose a module.
Practice: Click on a chapter to start the quiz. Select an option and hit Check to see if you are right.
Bookmark: Use the Bookmark button to save a question. Access it later from the sidebar menu.
Downloads: Go to the "Download PYQs" tab to access the Google Drive folders containing PDFs.
